# 数据清洗面板美化设计文档

**日期：** 2026-06-15
**类型：** 页面美化（重设计）
**状态：** 已批准，待实施

---

## 1. 背景

数据清洗面板（data-cleaning-taskpane）当前样式简陋，与项目中其他面板（如连接列工具）视觉风格差异较大。用户要求参照 Fluent UI 蓝色主调 + 引导卡片风格进行美化。

## 2. 设计方向

参照连接列工具（concat-taskpane）的设计风格：
- Fluent UI 蓝色主调
- 引导卡片（蓝边信息提示框）
- 蓝色边框输入控件

## 3. 页面结构

```
┌─────────────────────────────────────┐
│  🧹 数据清洗                         │  ← 蓝色标题 + SVG 图标
├─────────────────────────────────────┤
│  [引导卡片]                          │  ← 蓝边 3px，左侧蓝边提示
├─────────────────────────────────────┤
│  [标签栏]                            │  ← 横向标签页切换
│  修剪空格 | 删除空行 | 大小写转换 |    │
│  清除不可见 | 移除重复行              │
├─────────────────────────────────────┤
│  [参数设置区域]                       │  ← 动态切换
│    - 操作标签对应的参数控件            │
│    - 操作专属引导说明                  │
├─────────────────────────────────────┤
│  [预览表格]                           │
│    原值 | 清洗后（红绿对比）           │
├─────────────────────────────────────┤
│  [选区信息 + 刷新]                    │
│  [🚀 执行清洗]  [↩ 撤销]              │
│  [状态栏]                             │
└─────────────────────────────────────┘
```

## 4. 标签栏设计

| 状态 | 样式 |
|------|------|
| 未选中 | 灰底白字，无边框 |
| 选中 | 蓝底白字，顶部 3px 蓝色下划线 |
| Hover | 浅蓝背景 |

**标签列表：**
1. 修剪空格
2. 删除空行
3. 大小写转换
4. 清除不可见
5. 移除重复行

## 5. 引导卡片

每个标签显示对应操作的简短引导说明（左侧蓝边 3px，浅蓝背景）：

| 操作 | 引导说明 |
|------|---------|
| 修剪空格 | 1. 选中要清洗的单元格区域 2. 选择修剪模式 3. 查看预览后执行 |
| 删除空行 | 1. 选中数据区域 2. 选择判定方式 3. 查看预览后执行 |
| 大小写转换 | 1. 选中要转换的单元格 2. 选择转换模式 3. 查看预览后执行 |
| 清除不可见 | 1. 选中单元格区域 2. 选择清除类型 3. 查看预览后执行 |
| 移除重复行 | 1. 选中数据区域 2. 选择依据列和保留规则 3. 查看预览后执行 |

## 6. 参数设置区域

每个操作的参数控件样式与连接列工具保持一致：
- 标签文字 + 下拉框/输入框横向排列
- 2px 蓝色边框，聚焦时蓝色光晕
- 参数控件区域背景：`#fafafa`，浅灰边框

## 7. 预览表格

- 表头：灰底背景 (`#f0f0f0`)，加粗字体
- 原值列：浅红背景 (`#fff3f3`)
- 清洗后列：浅绿背景 (`#f3fff3`)
- 最多显示前 5 行，超出部分垂直滚动
- 最大高度：200px

## 8. 按钮样式

**执行按钮（主按钮）：**
- 背景：`#0078d4`，白字，无边框
- Hover：`#106ebe`
- Disabled：灰色背景 (`#ccc`)

**撤销按钮（次按钮）：**
- 背景：透明，蓝字，2px 蓝色边框
- Hover：浅蓝背景 (`#f0f7ff`)
- Disabled：灰色字，无边框

## 9. 状态栏样式

| 状态 | 样式 |
|------|------|
| idle | `display: none` |
| loading | 黄色背景 (`#fff3cd`)，深黄字 |
| success | 绿色背景 (`#d4edda`)，深绿字 |
| error | 红色背景 (`#f8d7da`)，深红字 |

## 10. HTML 结构变更

- 添加 `office.js` 引用（已在 fix 中完成）
- 添加 Fluent UI fabric CSS 引用

## 11. CSS 变更

新建/重写 `data-cleaning-taskpane.css`，参照 `concat-taskpane.css`：
- 蓝色主调 (`#0078d4`)
- 引导卡片样式（蓝边 + 浅蓝背景）
- 标签栏样式（横向排列 + 选中高亮）
- 参数区域样式（浅灰背景 + 蓝色控件）
- 预览表格样式（红绿对比列）
- 按钮样式（主按钮蓝底 + 次按钮边框）

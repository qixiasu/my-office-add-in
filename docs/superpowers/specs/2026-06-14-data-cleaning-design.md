# 数据清洗工具 — 设计文档

**日期：** 2026-06-14
**项目：** Excel 数据处理工具（My Office Add-in）
**状态：** 已批准

## 概述

为 Excel 工具箱新增一个「数据清洗」任务面板，提供 5 个高频数据清理操作：修剪空格、删除空行、大小写转换、清除不可见字符、移除重复行。采用统一面板 + 分区布局，一站式完成清洗工作流。

## 功能范围

| 操作 | 说明 | 参数 |
|------|------|------|
| 修剪空格 | 去除单元格多余空格 | 首尾空格 / 全部多余空格 / 全部移除 |
| 删除空行 | 删除空白行 | 完全空行 / 指定列为空 / 空值率 > N% |
| 大小写转换 | 文本大小写变换 | 大写 / 小写 / 首字母大写 / 驼峰式 |
| 清除不可见字符 | 移除非打印字符 | 换行符 / 制表符 / 零宽字符 / 全部 |
| 移除重复行 | 按条件去重 | 依据列（多选）· 保留首行 / 末行 |

## 实现方案

### 方案选择：方案 A — 统一面板 + 分区布局

三个备选方案的对比：

| 方案 | 描述 | 选择 |
|------|------|------|
| A | 单一任务面板，所有操作通过分区/选项切换 | ✅ 选此方案 |
| B | 工作流向导，可批量串联多个操作 | ❌ 对基础操作过于复杂 |
| C | 每个操作独立面板 | ❌ 功能区按钮爆炸 |

### 面板布局

```
┌─────────────────────────────┐
│ 📌 选区：[Sheet1!A1:C100]   │
│  ↻ 刷新选区                  │
├─────────────────────────────┤
│ 🔧 选择操作                  │
│  ○ 修剪空格  ○ 删除空行       │
│  ○ 大小写转换  ○ 清除不可见    │
│  ○ 移除重复行                │
├─────────────────────────────┤
│ ⚙️ 参数设置（动态切换）        │
├─────────────────────────────┤
│ 👁 预览（前 5 行对比）         │
│  原值 → 清洗后                │
├─────────────────────────────┤

└─────────────────────────────┘
```

## 数据流

```
用户选区 → getSelectionSummary → 预览（前端 JS 计算前5行）
                                       ↓
                              用户调参 → 实时更新预览
                                       ↓
                              确认执行 → 全量 JS 计算 → 一次性写回 Excel
                                       ↓
                              撤销快照（单次，关闭即清）
```

## 撤销机制

- 执行前备份原始 `values` 二维数组到内存
- 点击撤销 → 原始数据一次性写回
- 再次执行 → 新备份覆盖旧备份
- 关闭面板 → 缓存自动清空

## 边界情况处理

| 场景 | 处理 |
|------|------|
| 空选区 / 无数据 | 显示提示 + 刷新按钮 |
| 仅标题行 | 提示无内容行 |
| >10,000 行 | 弹出性能提醒 |
| 连续执行两次 | 只能撤销到上一次 |
| 跨会话撤销 | 不支持 |
| 选区切换 | 手动刷新，显示当前选区地址 |

## 核心接口（data-cleaning-utils.js）

所有函数均为纯函数，输入二维数组 → 输出新二维数组：

- `trimSpaces(values, mode)` — 修剪空格
- `removeEmptyRows(values, mode, columnIndex)` — 删除空行
- `convertCase(values, mode)` — 大小写转换
- `removeInvisible(values, mode)` — 清除不可见字符
- `removeDuplicates(values, keyColumns, keep)` — 去重

## 测试计划

预计 15-20 个 Jest 测试用例，覆盖每个函数的正常路径、边界值和空输入。

| 函数 | 关键测试场景 |
|------|-------------|
| `trimSpaces` | 首尾空格、多余中间空格、无空格、全空单元格 |
| `removeEmptyRows` | 完全空行、按列空、中间空行、正常数据 |
| `convertCase` | 大写/小写/首字母、中文不转换、混合文本 |
| `removeInvisible` | 换行符、制表符、零宽字符、正常文本不变 |
| `removeDuplicates` | 单列/多列、保留首/末、无重复、空值列 |
